üìã API Endpoints

POST /api/analyze/comprehensive
Complete product analysis using all available ML models. Supports image upload and returns detailed analysis including pricing, sustainability, and categorization.
POST /api/analyze/image
Advanced image analysis using CLIP and MobileNet models. Upload an image to get craft category predictions and visual features.
POST /api/analyze/pricing
Intelligent pricing prediction using XGBoost ensemble models. Considers craft type, region, materials, and sustainability factors.
POST /api/analyze/sustainability
Comprehensive sustainability analysis with CO2 impact, waste calculation, and certification recommendations.
GET /api/analytics
Real-time API usage statistics, performance metrics, and request analytics.
GET /api/history
Retrieve analysis history with pagination support. View past analyses and results.
GET /api/health
Health check endpoint to verify API status and model availability.
POST /api/feedback
Submit feedback for analysis results. Helps improve model accuracy through user feedback.
